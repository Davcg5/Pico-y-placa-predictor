# Pico-y-placa-predictor
## Description 
This program permits to know whether a car is under Pico y Placa restriction on the date and time provided by user 

## Instructions 
1. Run mainPredictor.py
2. Enter a plate number, the format for car is: AAA111 or AAA1111. The format for motorbike is: AA111A
3. Enter a date, the format accepted for the date is dd-mm-yyyy
4. Enter a time, the format accepted fot the time is H:M (24h format)
